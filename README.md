# ML Analysis of Chemical Compounds

Проект посвящён анализу химических соединений и построению моделей машинного обучения для прогнозирования их эффективности против вируса гриппа.

В работе рассматриваются три количественных показателя эффективности:
- IC50
- CC50
- SI

На основе числовых дескрипторов химических соединений решены задачи регрессии и классификации, выполнено сравнение моделей и проведён анализ качества.

## Цели проекта

Регрессия:
- предсказание IC50
- предсказание CC50
- предсказание SI

Классификация:
- IC50 > медианы
- CC50 > медианы
- SI > медианы
- SI > 8

## Структура проекта

ml-chemical-compounds/
- EDA.ipynb — разведочный анализ данных
- model_ic50.ipynb — регрессия для IC50
- model_cc50.ipynb — регрессия для CC50
- model_si.ipynb — регрессия для SI
- classification_ic50.ipynb — IC50 > медианы
- classification_cc50.ipynb — CC50 > медианы
- classification_si_median.ipynb — SI > медианы
- classification_si_gt8.ipynb — SI > 8
- report.pdf — аналитический отчёт
- README.md — описание проекта

## Использованные методы

Регрессия:
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

Классификация:
- Logistic Regression
- Random Forest Classifier

Использовался подбор гиперпараметров.

## Подготовка данных

- удаление служебного индексного столбца
- проверка на пропуски
- заполнение пропусков
- формирование целевых переменных
- разделение на обучающую и тестовую выборки
- устранение утечки данных

В задачах предсказания SI исключались признаки SI, IC50 и CC50.

## Основные результаты

- в регрессии лучшие результаты показал Random Forest
- качество классификации зависело от постановки задачи и распределения классов
- для задачи SI > median лучшая модель — Logistic Regression (по F1-score)
- для задачи SI > 8 лучшая модель — Random Forest (по F1-score)
- задача SI > 8 сложнее из-за дисбаланса классов
